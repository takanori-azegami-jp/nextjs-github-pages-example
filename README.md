# nextjs-github-pages-example
GitHub PagesにNext.js をデプロイする

## 1.Node.jsインストールする

## 2.Next.jsを設定
```bash
$ mkdir [リポジトリ名]
$ cd [リポジトリ名]
$ npm init -y
$ npm install --save react react-dom next
$ mkdir pages
```
